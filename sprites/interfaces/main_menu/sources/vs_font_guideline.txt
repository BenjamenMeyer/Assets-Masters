Create VS Text
- In GIMP 2.00: create selection:
  Layer > Text to Selection > Text to Selection
- In GIMP 2.10: create selection:
  Layer > Text to Path
  Select > From Path
- Create new layer
- Rename layer to _glow
- Select layer
- Select > Grow = 2
- Select > Feather = 6
- Fill
- Stack layer below font
